# Pflegekörbchen

Informationsseite für monatliche Pflegehilfsmittel nach § 40 SGB XI.

## Dateien

- `index.html` – Hauptseite (One-Pager)
- `impressum.html` – Impressum
- `datenschutz.html` – Datenschutzerklärung
- `agb.html` – Allgemeine Geschäftsbedingungen

## Vor dem Live-Gang

1. In `index.html`: `DEIN-EXTERNER-LINK-HIER` durch den echten Link zum Antragspartner ersetzen
2. In `impressum.html`: Platzhalter mit echten Angaben füllen
3. `datenschutz.html` und `agb.html` mit rechtlich geprüften Texten ersetzen (z.B. via e-recht24.de)

## Hosting

Gehostet über Vercel, verbunden mit diesem GitHub Repository. Änderungen werden automatisch deployed.
